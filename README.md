# Array Manager

Interactive JavaScript app for learning array methods.

## Features

- Add, remove, sort, reverse array elements
- Filter even numbers
- Calculate sum
- Responsive design

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)

## Array Methods Used

`.push()` `.pop()` `.shift()` `.reverse()` `.sort()` `.filter()` `.reduce()`
